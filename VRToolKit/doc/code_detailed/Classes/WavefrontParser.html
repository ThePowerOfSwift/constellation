Overview

This class can parse basic wavefront files (.obj)

For now it is very slow, and is not very usable to load complicated objects on demand prefer the .h method in that case.

gLfloatArrayFromArray:usingIndexArray:

This method creates an array merging an index and some data.

+ (GLfloat *)gLfloatArrayFromArray:(NSArray *)array usingIndexArray:(NSArray *)indexArray
Parameters
array

The array containing datas

indexArray

The array containig the indexes

Return Value

an array created from the datas and the index

Discussion

For instance if the array is: 1,2,7,9,0 and the index array is: 0,0,4 the array returned will be 1,1,9. This method is used to have an array of vertices corresponding to the faces of the 3D object.

gLTexturesArrayFromArray:usingIndexArray:

This method creates an array merging an index and some data.

+ (GLfloat *)gLTexturesArrayFromArray:(NSArray *)array usingIndexArray:(NSArray *)indexArray
Parameters
array

The array containing datas

indexArray

The array containig the indexes

Return Value

an array created from the datas and the index

Discussion

This method does the same job as (GLfloat *)gLfloatArrayFromArray:(NSArray *)array usingIndexArray:(NSArray *)indexArray but for the texture coordinate datas. We also do basic calculation to convert texture UV coordinates in ST coordinates so that openGL can understand them. This method is used to have an array of texture coordinates corresponding to the faces of the 3D object.
