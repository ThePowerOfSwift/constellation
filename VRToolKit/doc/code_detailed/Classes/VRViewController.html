Overview

This class is the core of the Application.

It initializes the capture of frames from the camera, send them to the ARToolKitPlusWrapper, receives the result of the marker detection and pass on those results to the EAGLView.

initCapture

This methods initialize the capture session.

- (void)initCapture
Discussion

It sets up the AVCaptureDeviceInput instance, the AVCaptureVideoDataOutput instance and create an AVCaptureSession which links the input and the output. The format chosen for the capture is kCVPixelFormatType_32BGRA because for now ARToolKitPlusWrapper want BGRA encoded frames. The queue used is not the main queue, so all the capture and detection process is done in a separate thread.
