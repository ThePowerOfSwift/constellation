ARToolKitPlusWrapperDelegate Protocol Reference

Declared in ARToolKitPlusWrapper.h

Overview

This protocol describes the delegates methods that the delegate of an ARToolKitPlusWrapper instance must implement.

An ARToolKitPlusWrapper instance use the delegate pattern to send back datas when it is initialized and when a marker is or is not detected. This protocol describes the methods that the ARToolKitPlusWrapper instance calls on its delegate. For now only single marker detection is implemented.
